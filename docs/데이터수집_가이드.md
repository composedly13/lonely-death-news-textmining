# 데이터 수집 가이드 — 2코퍼스 갭 분석 (기말과제)

빅카인즈([bigkinds.or.kr](https://www.bigkinds.or.kr))에서 아래 2개 코퍼스를 수집해 **갭(gap)** 을 비교한다.
두 코퍼스는 **같은 기간(2025-06-01 ~ 2026-06-01)·같은 필드(제목+본문)** 로 맞춰 *시점을 통제*한다.
(시기 차이가 아니라 **담론 차이**를 비교하기 위함)

> **분석 가설**: 뉴스 담론(A)은 시니어를 *복지 수혜·위기 대상*으로, 서비스 담론(B)은 *능동적 사용자·소비자*로
> 다룬다 — 이 갭을 객관 지표로 정량 입증한다.

## 코퍼스 A — 고독사·독거노인 담론

| 항목 | 값 |
|---|---|
| 검색어(OR) | `독거노인, 고독사, 1인 고령가구, 노인 고립, 무연고 사망` |
| 기간 | 2025-06-01 ~ 2026-06-01 |
| 필드 | 제목 + 본문 |
| 저장 위치 | `data/raw/A/*.xlsx` (2만건 제한으로 6개월씩 2파일) |
| 정제 후 | **20,079건** |

## 코퍼스 B — 스마트홈·시니어 서비스

| 항목 | 값 |
|---|---|
| 검색어(OR) | `스마트홈, 실버테크, 고령친화, 고령친화산업, 돌봄로봇, AI돌봄, 스마트돌봄, 스마트경로당, 시니어케어, 시니어비즈니스, 액티브시니어, 실버산업, 실버타운, 에이지테크, 시니어테크, 디지털헬스케어, 노인맞춤돌봄, 비대면돌봄, 안부확인, 응급안전안심, 효돌, 반려로봇` |
| 기간 | 2025-06-01 ~ 2026-06-01 (A와 동일) |
| 필드 | 제목 + 본문 |
| 저장 위치 | `data/raw/B/*.xlsx` |
| 원본 → 정제 → 시니어필터 | 13,125 → 12,886 → **5,059건** |

### 코퍼스 B 시니어 후처리 필터 (분석 코드에서 자동 적용)

OR 검색이라 `스마트홈`(건설·환기), `응급안전`(폭염·재난), `수혜`(증권 '수혜주') 등
**비(非)시니어 기사가 섞인다.** 객관·재현 가능하게 걸러내기 위해,
분석 노트북이 **제목·본문에 아래 시니어 키워드가 하나라도 있는 기사만** 남긴다.

```
노인 · 고령 · 시니어 · 어르신 · 독거 · 실버 · 경로당 · 치매 · 요양
```

> 빅카인즈 상세검색에서 위 OR 키워드에 `AND (노인 OR 고령 OR 시니어 OR 어르신 OR 독거 OR 실버)` 를
> 직접 걸면 수집 단계에서 동일하게 좁힐 수 있다(후처리 손실 최소화).

## 비교 설계 (객관성 확보 장치)

| 장치 | 내용 |
|---|---|
| **시점 통제** | A·B 동일 기간(1년)·동일 필드 |
| **규모 균형** | A를 B와 같은 5,059건으로 **무작위 다운샘플(random_state=42)** → 1:1 비교. 전체 A(2만)로도 재확인해 "크기 무관" 명시 |
| **교차 중복** | A∩B = 895건(양쪽 검색에 걸린 기사)은 **그대로 유지** — 갭을 과장하지 않는 보수적 처리 |
| **정규화 지표만 사용** | 토픽 비중(%), 문서당 출현율, log-odds ratio, 감성 비율 — 코퍼스 크기에 영향받지 않음 |

## 공통 전처리 규칙

- **`뉴스 식별자`는 반드시 문자열(`dtype=str`)로 읽는다.** 26자리 ID를 숫자로 읽으면 정밀도 손실로 중복제거가 오작동한다.
- `분석제외 여부`가 `중복`/`예외`/`중복, 예외`인 행 제거 → 제목·본문 기준 추가 중복 제거.
- 전처리: URL·이메일·비한글 제거 → Okt 명사 추출 → 복합 키워드(독거노인 등) 재결합 → 행정 상투어 불용어 제거.

## 객관적 갭 지표 (5종)

1. **토픽 비중 비교** — A·B 각각 LDA → 토픽 분포 대조 (공통/고유 주제)
2. **키워드 변별도** — log-odds ratio로 각 코퍼스에 통계적으로 특징적인 단어 추출
3. **호명 어휘 정량화** — '수혜·대상화'(지원·보호·취약·돌봄·복지…) vs '주체·소비'(이용·구매·제품·서비스·기능·사용자…) 어휘의 문서당 출현율 비교
4. **감성 비교** — 동일 KNU 감성사전으로 A vs B 평균감성·분포 대조
5. **토픽 유사도** — 두 코퍼스 토픽 간 코사인 유사도 매트릭스 (겹치는/고유 주제 시각화)
