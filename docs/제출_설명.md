## 3. 분석 개요

| 항목 | 내용 |
|---|---|
| 분석 대상 | 빅카인즈 뉴스 (2025-06-01 ~ 2026-06-01, 1년 전수) |
| 수집 키워드(OR) | 독거노인 · 고독사 · 1인 고령가구 · 노인 고립 · 무연고 사망 |
| 분석 문서 수 | **20,079건** (원본 20,521건 → 식별자·제목/본문 중복 및 예외 제거 후) |
| 분석 파이프라인 | 빈도·워드클라우드 → LDA 토픽모델(k=6) → 동시출현 네트워크 → 감성분석(KNU 사전) → **토픽×감성 교차** |

**연구 질문(RQ)**
1. (RQ1) 고독사·독거노인 보도는 어떤 주제로 구성되며, 어느 프레임이 우세한가?
2. (RQ2) 보도 전반의 감성은 긍정적인가 부정적인가?
3. (RQ3) 어떤 주제를 다룰 때 어조가 달라지는가? (토픽 × 감성)

## 4. 핵심 결과

> **한 줄 요약 — 고독사·독거노인은 사회적 '위험' 이슈지만, 언론은 이를 주로 복지·나눔이라는 '시혜적·긍정적' 프레임으로 소비한다.**

| 분석 | 핵심 수치 | 해석 |
|---|---|---|
| **토픽 (RQ1)** | 시혜·복지 프레임 **약 50%** > 위기 프레임 **약 38%** (나머지 12% 노인의료·돌봄) | 위기보다 시혜·활동홍보 담론이 우세 |
| **감성 (RQ2)** | 긍정 **61.2%** / 중립 15.5% / 부정 **23.3%**, 평균 점수 **+1.47** | 죽음·고립 이슈인데도 뚜렷한 긍정 편향 |
| **토픽×감성 (RQ3)** | 부정 비율이 높은 주제(약 27~30%)는 재난안전대응·고독사예방·노인의료돌봄 등 *고통을 직접 다루는 주제*뿐. **단 이들조차 평균 감성은 순(純)긍정(+0.93)** | 부정 정서는 '위험을 직접 말하는' 좁은 영역에 갇혀 있고, 어디서도 지배적 어조가 되지 못함 |
| **네트워크** | 중심어 = `복지·가구·취약·계층·독거노인` (49노드·294엣지) | 독거노인이 주로 *복지 수혜 대상*으로 호명됨 |

네 가지 분석이 **'복지 대상으로서의 독거노인'이라는 하나의 그림**으로 수렴합니다.

## 5. 방법론 요점

- **전처리**: URL·이메일·비한글 제거 후 Okt 명사 추출. `독거노인` 등 복합 키워드는 Okt가 쪼갠 조각을 재결합해 보존. 행정·보도자료 상투어(`기자/제공/추진/실시` 등)는 불용어 제거하되 분석 핵심어(`복지/가구/취약/계층` 등)는 보존.
- **LDA**: `filter_extremes(no_above=0.5)`로 과빈출어 제거. coherence(c_v)는 환경마다 흔들리고 큰 k를 과대평가하는 경향이 있어, k=6과 k=12가 사실상 동률인 점·k=12가 지명/잡음으로 과분할되는 점을 근거로 **해석 가능성 기준 k=6으로 고정**.
- **네트워크**: 빈도 상위 50개 단어 중 같은 기사에 300회 이상 공출현한 쌍을 연결, 연결·매개 중심성 산출.
- **감성**: KNU 한국어 감성사전(단어형). 명사만으로는 감성어를 못 잡으므로 `Okt.pos(stem=True)`로 형용사·동사 어간까지 복원해 매칭.

## 6. 분석의 한계 (보고서에도 명시)

- 감성 점수는 **사전 기반**이라 문맥 부정("외롭지 **않다**")·반어를 반영하지 못함 → 절대값보다 **주제 간·월별 상대 비교**로 해석.
- 긍정 편향은 실제 보도 성향일 수도, 사전 특성(봉사·지원 어휘가 긍정)일 수도 있어 단정은 피함.
- 빅카인즈 키워드 OR 검색 기반이라 표본은 해당 키워드 보도에 한정됨.
- LDA 토픽 ID 순서는 실행 환경에 따라 달라질 수 있으나, 출력 키워드로 라벨 매핑을 확인할 수 있으며 **결론의 수치는 ID 매핑과 무관하게 성립**.
